# GamesAI

A workspace for browser games built with TypeScript and Canvas.

## Games

- [Gantos](./Gantos) - published at https://tharak.github.io/gantos/Gantos/
- [GiantGorilla](./GiantGorilla) - published at https://tharak.github.io/GiantGorilla/

## Structure

Each game lives in its own folder with its own Vite project, dependencies, and build scripts.
